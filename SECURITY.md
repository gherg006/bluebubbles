# Security policy

BlueBubbles is under active pre-release development and does not yet have a
supported production release.

Do not disclose suspected vulnerabilities in a public issue. Report them
privately to the repository owner with a description, affected version,
reproduction steps, and potential impact. Do not include real credentials,
private keys, message content, or personal data in a report.

The owner will acknowledge a report, assess its severity, coordinate a fix,
and publish appropriate release notes before public disclosure.
